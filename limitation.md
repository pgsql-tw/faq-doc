# PostgreSQL 資料庫可以有多大？[^1]

| **限制** | **範圍** |
| :--- | :--- |
| 資料庫（Database）大小限制 | 無限制 |
| 最大資料表（Table） | 32 TB |
| 最大資料列（Row） | 1.6 TB |
| 最大欄位（Field）內容 | 1 GB |
| 資料表中最多可以有幾個資料列 | 無限制 |
| 資料表中最多可以有幾個欄位 | 250 - 1600 （與欄位型別有關） |
| 資料表最多可以有幾個索引（Index） | 無限制 |

---

[^1]:  [PostgreSQL: About](https://www.postgresql.org/about/)
